## Usage

A button with an icon and label:

```javascript
<button>
  <div style={{ display: 'flex', alignItems: 'center' }}>
    <Icon size={16} variant="SUCCESS" />
    <span style={{ margin: '1px 0 0 1px' }}>Success</span>
  </div>
</button>
```
